# Sort-Algorithms-with-Tkinter-Python
A simple program made with Python Using Tkinter library for the interface

Implementation of sorting algorithms as following:
  - Shell Sort
  - Insert Sort
  - Bubble Sort
  - Merge Sort
  - Quick Sort
  - Bucket Sort
  - Radix Sort
  - Heap Sort
  - Count Sort
  - Bin Sort
  - Randomized Select Sort   --->  (still with failures)
  - Stooge Sort
